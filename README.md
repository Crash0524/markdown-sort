> Tip: 支持选中markdown列表的分级排序。对非列表内容保持原顺序和位置。


## 如何使用

例：

+ 选中需要排序的列表，命令行中输入`Sort Markdown List`
+ 选中需要排序的列表，右键`Sort Markdown List`
